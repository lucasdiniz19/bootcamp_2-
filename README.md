# Vansmind CLI 🚐 - Bootcamp Delivery

## 🎯 Problema Real
Muitos motoristas de transporte escolar operam sem saber exatamente a margem de lucro de cada rota, considerando o alto custo do diesel e as distâncias percorridas. Esta ferramenta resolve essa dor ao calcular instantaneamente a viabilidade financeira.

## 🚀 Tecnologias e Boas Práticas
- **Node.js**: Runtime principal.
- **Jest**: 3 testes automatizados (Caminho feliz, erro e limite).
- **ESLint**: Garantia de qualidade de código.
- **Versionamento Semântico**: 1.0.0 (MAJOR.MINOR.PATCH).
